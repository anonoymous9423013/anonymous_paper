# Supplementary Materials for "Evaluating the Impact of Flaky Simulators on Testing Autonomous Driving Systems" paper

This directory contains supplementary materials for the paper "Evaluating the Impact of Flaky Simulators on Testing Autonomous Driving Systems". The materials include:

- `results.pdf`: a PDF file with the results for three different thresholds of 5%, 10%, and 20%, related to RQ1 and RQ2 in the paper.

The `results.pdf` file contains all the experimental results generated for the research questions RQ1 and RQ2 in the paper. The results are provided for three different thresholds of 5%, 10%, and 20%, and include detailed plots and tables.
